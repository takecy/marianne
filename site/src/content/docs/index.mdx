## What is marianne?

`marianne` is a lightweight desktop app for annotating images — rectangles, arrows, text, and pixelation — designed to run **fully offline**. Screenshots never leave your machine, so it is safe for sensitive content.

This site is the **contributor-facing** documentation. End-user instructions (download, install, keyboard shortcuts) live in the repository README.
