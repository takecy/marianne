This page is a personal note from the author about why Marianne exists, where it draws inspiration from, and how it relates to the names it borrows.

## A tribute to Skitch

[Skitch](https://en.wikipedia.org/wiki/Skitch) was created by [Plasq](https://www.plasq.com/) in 2007 and acquired by [Evernote](https://evernote.com/) in 2011. For many years it was the go-to lightweight tool on macOS for screenshot annotation: rectangles, arrows, text, pixelation, and a few colors — nothing more, nothing less.

In January 2016, Evernote [announced](https://www.pcworld.com/article/418827/evernote-drops-dead-weight-with-demise-of-clearly-pebble-app-and-most-skitch-versions.html) that Skitch's Windows, iOS, and Android editions would be discontinued. The Mac app continued to receive limited support afterward, but it now sits unmaintained as a legacy Intel-only binary.

As of 2026, Apple Silicon Macs can still run Intel binaries via [Rosetta 2](https://support.apple.com/en-ca/102527), so Skitch for Mac technically still works for now. However, Apple has [officially announced that Rosetta 2 will be deprecated in future macOS releases](https://developer.apple.com/documentation/apple-silicon/about-the-rosetta-translation-environment/), and in the near future Skitch will stop working.

## Why I built Marianne

I had been a Skitch devotee for years — using it on every document and presentation. Its minimal feature set and snappy performance were exactly what I needed. Then one day macOS popped up a warning that Skitch would soon stop working, and I was devastated. I tried a handful of alternatives, but none of them matched my workflow the way Skitch did. So I decided to build a Mac app of my own — one that runs natively on Apple Silicon and behaves the way Skitch always did.

## About the name

The app's name **Marianne** comes from the real name of [Ms. Goldenweek](https://one-piece.com/character/Ms_Goldenweek/index.html), a painter character from the manga ONE PIECE.
