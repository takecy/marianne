このページは作者から読者へ向けた個人的なメモ。Marianne がなぜ存在するか、どこからインスパイアされているか、借りている名前との関係性を記した。

## Skitch へのトリビュート

[Skitch](https://ja.wikipedia.org/wiki/Skitch) は [Plasq](https://www.plasq.com/) により 2007年に作られ、2011年に [Evernote](https://evernote.com/) が買収した。長年 macOS で軽量に使えるスクリーンショット注釈ツールの定番で、矩形・矢印・テキスト・ピクセル化・少数の色 — それだけが揃っていた。

2016年1月、Evernote は [Skitch の Windows / iOS / Android 版をディスコン](https://www.pcworld.com/article/418827/evernote-drops-dead-weight-with-demise-of-clearly-pebble-app-and-most-skitch-versions.html) した。Mac 版はその後もしばらくサポートが続いたが、現在はメンテナンスされておらず legacy Intel-only バイナリのままの状態。

2026年現在、Apple Silicon Mac でも [Rosetta 2](https://support.apple.com/ja-jp/102527) 経由で Intel バイナリは動作するため、Skitch Mac 版は今のところ動く。しかし、Apple は今後の macOS で [Rosetta 2 を deprecate していくと公式に発表](https://developer.apple.com/documentation/apple-silicon/about-the-rosetta-translation-environment/)していて、近い将来Skitchは動かなくなる。

## なぜ作ったか

作者は長年の Skitch 愛用者。あらゆる資料やプレゼンで愛用し、必要十分な機能と軽快な動作が大のお気に入りだった。しかし、ある日 macOS に、 「もうすぐSkitch を動かせなくなる」とポップアップが表示され絶望し、代替になるアプリを探していくつか試してみたが、私の用途に必要十分なアプリは見つからなかった。そこで、Apple Silicon ネイティブで動き Skitch と同じように振る舞うmacアプリを作ることにした。

## 名前について

アプリ名 **Marianne** は、漫画 「ONE PIECE」に登場する画家「[ミス・ゴールデンウィーク](https://one-piece.com/character/Ms_Goldenweek/index.html)」の本名から。


## 商標・著作権の表記

- **Skitch** は権利者 (Evernote / Plasq 系譜) の商標。Marianne は Plasq・Evernote のいずれとも提携・公式承認関係はない。
- **「ONE PIECE」** は集英社の商標で、原作は尾田栄一郎による。Marianne は集英社・尾田栄一郎のいずれとも提携・公式承認関係はない。
- 本ページに記載のその他の製品名は各権利者の所有物。

本ページは Marianne を可能にしたインスピレーションへの敬意を表し、名前の由来を明確にするために存在する。Skitch に長寿を。ONE PIECEが300巻まで続きますように。
