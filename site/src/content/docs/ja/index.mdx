## marianne とは

`marianne` は画像にアノテーション (矩形 / 矢印 / テキスト / モザイク) を付与する軽量デスクトップアプリで、**完全オフライン**で動作する。スクリーンショットが端末外に出ることは一切ないため、機密情報を含むコンテンツも安全に扱える。

このサイトは **contributor 向け** のドキュメント。エンドユーザー向けの情報 (ダウンロード、インストール、ショートカット) はリポジトリ README に記載している。
