What is Marianne?

Section titled “What is Marianne?”

Marianne is a lightweight desktop app for annotating images — rectangles, arrows, text, pixelation, and image cropping — designed to run fully offline. Screenshots never leave your machine, so it is safe for sensitive content.

Marianne is a tribute to Skitch, built as a forward-looking replacement for the unmaintained legacy app. → Read the tribute
