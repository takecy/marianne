トリビュートと由来

このページは作者から読者へ向けた個人的なメモ。Marianne がなぜ存在するか、どこからインスパイアされているか、借りている名前との関係性を記した。

Skitch へのトリビュート

Section titled “Skitch へのトリビュート”

Skitch は Plasq により 2007年に作られ、2011年に Evernote が買収した。長年 macOS で軽量に使えるスクリーンショット注釈ツールの定番で、矩形・矢印・テキスト・ピクセル化・少数の色 — それだけが揃っていた。

2016年1月、Evernote は Skitch の Windows / iOS / Android 版をディスコン した。Mac 版はその後もしばらくサポートが続いたが、現在はメンテナンスされておらず legacy Intel-only バイナリのままの状態。

2026年現在、Apple Silicon Mac でも Rosetta 2 経由で Intel バイナリは動作するため、Skitch Mac 版は今のところ動く。しかし、Apple は今後の macOS で Rosetta 2 を deprecate していくと公式に発表していて、近い将来Skitchは動かなくなる。

なぜ作ったか

Section titled “なぜ作ったか”

作者は長年の Skitch 愛用者。あらゆる資料やプレゼンで愛用し、必要十分な機能と軽快な動作が大のお気に入りだった。しかし、ある日 macOS に、 「もうすぐSkitch を動かせなくなる」とポップアップが表示され絶望し、代替になるアプリを探していくつか試してみたが、私の用途に必要十分なアプリは見つからなかった。そこで、Apple Silicon ネイティブで動き Skitch と同じように振る舞うmacアプリを作ることにした。

名前について

Section titled “名前について”

アプリ名 Marianne は、漫画 「ONE PIECE」に登場する画家「ミス・ゴールデンウィーク」の本名から。
