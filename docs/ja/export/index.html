注釈付き画像の書き出し

Marianne は注釈付きキャンバスをフラットな PNG として書き出す。2 つの出力先がサポートされており、出力内容は同じで使い方だけが異なる。

ファイル保存

Section titled “ファイル保存”

  1. Cmd / Ctrl + Shift + S を押すか、ツールバーの 保存 をクリックする。

  2. ネイティブ保存ダイアログが開く。ソース画像と同じディレクトリがデフォルトの保存先となる (drag-drop または Open With で読み込んだ場合)。それ以外の場合は書類フォルダにフォールバックする。

  3. ファイル名を確認し 保存 をクリック。画像の自然解像度で PNG が書き出される。

出力ファイルには元画像と全シェイプがラスタライズされて含まれる。解像度は元画像の自然サイズと一致し、拡大・縮小は行わない。

システムクリップボードへコピー

Section titled “システムクリップボードへコピー”

  1. Cmd / Ctrl + Shift + C を押すか、ツールバーの コピー をクリックする。

  2. 注釈付き PNG が macOS のシステムクリップボードに載る。

  3. 任意のアプリ (Slack / GitHub Issue / Mail など) にペーストする。

ファイルシステムを介さない — ディスクに残す必要がない「サッと共有」ワークフローに最適。

`Cmd + C` が 2 種類ある理由

Cmd + Shift + C は 注釈付き画像 をシステムクリップボードに載せる。 プレーンな Cmd + C (Shift なし) は別経路で、選択中の シェイプ を Marianne の内部クリップボードへコピーしてアプリ内複製に使う。全マッピングは キーボードショートカット を参照。

書き出される内容

Section titled “書き出される内容”
  • 元画像 (自然解像度)。
  • キャンバス上に表示されている全ての注釈 — 矩形・矢印・テキスト・モザイク。
  • モザイク領域は出力にラスタライズされて焼き付けられる。書き出された PNG からモザイクの下のピクセルを復元することはできない (機密情報を含むスクリーンショットを安全に共有できる)。

モザイクはエクスポート時点で不可逆

キャンバス上のモザイクは非破壊 (モザイク・シェイプを移動・削除できる) だが、 一度エクスポートすると PNG にピクセル化が焼き付けられる。エクスポート済み ファイルを受け取った人は、下のピクセルを復元できない。

出力仕様

Section titled “出力仕様”
項目値
フォーマットPNG
色空間sRGB
解像度元画像の自然ピクセルサイズ (スケーリングなし)
アルファチャンネル元画像のものを保持
圧縮標準 PNG (可逆)

別のフォーマットや解像度が必要な場合は、まず PNG で書き出してから別ツール (例: sips, ImageMagick) で変換すること。
