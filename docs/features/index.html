Features

Marianne keeps the annotation surface small on purpose. Four annotation shapes plus an image crop tool, eight colors, four stroke widths, and a handful of gestures cover almost every screenshot review you will ever do.

Annotation tools

Section titled “Annotation tools”

Rectangle

Outline a region with an unfilled rectangle. Useful for “look here” callouts. Stroke width is configurable (see below).

Arrow

Draw a Skitch-style arrow with a thick outline and drop shadow so it reads against busy backgrounds. Drag the endpoints later to redirect.

Text

Drop a text label anywhere on the canvas. Resize handles on the corners scale the font, and double-click reopens the editor.

Mosaic

Pixelate a rectangular region to redact faces, names, or other sensitive content. The pixelation is baked into the exported PNG so the original pixels cannot be recovered.

Switch tools from the left sidebar or with single-key shortcuts (v / a / r / t / m / x) — see Keyboard shortcuts.

Image crop

Section titled “Image crop”

The crop tool (sidebar icon or shortcut x) replaces the loaded image with a freshly cut region. On entry a default selection (the centre 80% of the image) appears with eight resize handles; drag the body to move it, drag a handle to resize. Enter or the Apply button commits, Escape or Cancel discards.

Caution

Crop is treated like loading a new image: the undo / redo history is reset on commit. There is no way to undo back to the pre-crop canvas — save first if you need a backup. Annotations that fall fully outside the crop region are deleted; rectangles, mosaics, and arrows that straddle the boundary are clipped to the new image bounds (arrows that miss the crop rectangle entirely are deleted, not clamped to the edge).

Color presets

Section titled “Color presets”

Eight high-contrast colors apply to rectangles, arrows, and text. Mosaic has no color (pixelation isn’t tinted).

Text shapes pick a contrasting outline automatically so red / blue / black text stays readable on light or dark backgrounds.

Rectangle stroke widths

Section titled “Rectangle stroke widths”

Rectangles support four preset stroke widths. The default is thick, matching the legacy fixed-width behavior.

PresetWidth (px)
thin6
medium12
thick18 (default)
extraThick28

Arrows have a single fixed stroke profile. Mosaic blocks use a fixed natural-pixel size, sized for redaction at typical screenshot resolutions.

Quality-of-life gestures

Section titled “Quality-of-life gestures”

Option + drag to duplicate

Section titled “Option + drag to duplicate”

Select a shape, then hold Option (Alt) while dragging. The original stays put, a clone follows your cursor, and on mouse-up the clone is placed at the drop point. Selection automatically jumps to the new clone.

Tip

The gesture only fires when Option is held at both the start and the end of the drag. Releasing Option mid-drag turns it back into a plain move — handy when you change your mind.

In-app shape clipboard

Section titled “In-app shape clipboard”

Cmd/Ctrl + C with a shape selected copies it to Marianne’s internal clipboard. Cmd/Ctrl + V pastes it at a small offset from the original, so repeated pastes form a visible stack.

The internal clipboard does not touch the OS clipboard, which means:

  • You can keep an OS image on the clipboard ready to paste as a new canvas image while still copy/pasting shapes inside Marianne.
  • If the internal clipboard is empty, Cmd/Ctrl + V falls back to the OS image paste — see Image input.

Undo / redo

Section titled “Undo / redo”

Up to 50 steps of history are kept per image. Loading a new image resets the history (you cannot undo back to the previous image’s annotations).

Status bar

Section titled “Status bar”

The bottom-of-window status bar shows:

  • Left side: where the current image came from — full file path (drag-drop / Finder “Open With” / File → Open… menu), or pasted from clipboard for paste-loaded images.
  • Right side: the image’s natural dimensions (e.g. 1920 × 1080).

Use it to confirm at a glance which screenshot you are annotating, especially when juggling several at once.
