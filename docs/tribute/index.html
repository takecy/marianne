Tribute & Origins

This page is a personal note from the author about why Marianne exists, where it draws inspiration from, and how it relates to the names it borrows.

A tribute to Skitch

Section titled “A tribute to Skitch”

Skitch was created by Plasq in 2007 and acquired by Evernote in 2011. For many years it was the go-to lightweight tool on macOS for screenshot annotation: rectangles, arrows, text, pixelation, and a few colors — nothing more, nothing less.

In January 2016, Evernote announced that Skitch’s Windows, iOS, and Android editions would be discontinued. The Mac app continued to receive limited support afterward but has since become an unmaintained legacy Intel-only binary.

As of 2026, Apple Silicon Macs can still run Intel binaries via Rosetta 2, so Skitch for Mac technically still works for many users. However, Apple has stated that Rosetta 2 will be deprecated in future macOS releases. When that happens, every long-time Skitch user will be looking for a replacement.

Marianne is that replacement, built before the deprecation lands.

Why I built Marianne

Section titled “Why I built Marianne”

I have been a Skitch user for years. Faced with the inevitability of Skitch breaking on a future macOS release, I built Marianne so that — when the day comes — I have a tool I trust that runs natively on Apple Silicon, is maintained, and behaves the way Skitch always did.

What Marianne inherits from Skitch

Section titled “What Marianne inherits from Skitch”

The design philosophy is intentionally Skitch-like:

  • Four tools, eight colors, four stroke widths. A small, fixed surface beats a kitchen-sink toolbar.
  • Fully offline. No telemetry, no cloud sync, no sign-in. Screenshots never leave your machine. The only outbound call is the auto-update check against GitHub Releases.
  • Clipboard-first UX. Paste an image. Annotate. Copy back or save as PNG. That’s it.
  • macOS native. Built with Tauri v2 — a native shell that integrates with system clipboard, dialogs, and updater.

What Marianne intentionally changes

Section titled “What Marianne intentionally changes”
  • Adds a mosaic tool that rasterises into the exported PNG, so the underlying pixels are not recoverable from the exported file. This is for visually obscuring screenshot regions — for high-stakes secrets like passwords or full tokens, consider an additional pre-export tool that overwrites pixels with a solid color.
  • License is PolyForm Noncommercial 1.0.0, not MIT. The original Skitch was distributed as free-but-proprietary software; Marianne adopts a source-available license that mirrors the spirit of “free for individuals, not for resale or proprietary product integration.” See the LICENSE file for the full terms.

The name “Marianne”

Section titled “The name “Marianne””

The project is named after Marianne — better known as Ms. Goldenweek — a painter character from the manga ONE PIECE by Eiichiro Oda (published by Shueisha).

Choosing a painter’s name for an image annotation app was a small piece of personal whimsy — the kind of detail that makes a side project feel like home.

No ONE PIECE artwork is used

Marianne does not embed any ONE PIECE character art, official logos, or screenshots. The name is a textual reference only.
