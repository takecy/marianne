Keyboard shortcuts

Cmd is shown for macOS — on other platforms read it as Ctrl. All shortcuts are suppressed while a text-annotation overlay or any native input / textarea has focus, so the browser’s own editing keys keep working in those fields.

Export

Section titled “Export”
ShortcutAction
Cmd + Shift + SSave the annotated PNG to a file (opens the native save dialog)
Cmd + Shift + CCopy the annotated PNG to the system clipboard

See Export for the full export workflow and the difference between the two paths.

Edit history

Section titled “Edit history”
ShortcutAction
Cmd + ZUndo (up to 50 steps)
Cmd + Shift + ZRedo
Delete / BackspaceDelete the selected shape (select mode only)

History resets when you load a new image — you cannot undo back to annotations on a previous image.

Shape clipboard

Section titled “Shape clipboard”
ShortcutAction
Cmd + CCopy the selected shape to Marianne’s internal clipboard (select mode only)
Cmd + VPaste from the internal clipboard at a small offset; falls back to the OS image-paste flow when the internal clipboard is empty

Why two `Cmd + C`s?

Plain Cmd + C operates on Marianne’s internal shape clipboard — it never touches the OS clipboard. Cmd + Shift + C is the unrelated export path that puts the annotated image on the system clipboard for pasting into Slack, chat apps, or other tools.

Tool switching

Section titled “Tool switching”
ShortcutTool
vSelect
aArrow
rRectangle
tText
mMosaic
xCrop

Tool shortcuts are modifier-less single keys (no Cmd / Ctrl / Shift / Alt). They are ignored when no image is loaded, which keeps v etc. from intercepting key presses on the empty start screen.

Crop mode

Section titled “Crop mode”

While the crop tool is active, two additional keys handle the selection:

ShortcutAction
EnterApply the crop (same as the Apply button)
EscapeCancel and return to the select tool (same as Cancel)

See Features → Image crop for the behaviour and history-reset semantics.

Discoverability tips

Section titled “Discoverability tips”
  • Hover any toolbar button to see its shortcut in the tooltip.
  • The status bar near the bottom shows the active tool icon for quick orientation.
  • Need to remember just three? r for rectangle, Cmd + Z to undo, Cmd + Shift + S to save. Everything else builds on those.
